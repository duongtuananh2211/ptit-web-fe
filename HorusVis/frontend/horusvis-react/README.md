# HorusVis React

Frontend scaffold for HorusVis using `npm` + `Vite` + `React` + `TypeScript`.

Current foundations included in this scaffold:

- `react-router-dom` route shell for Login, Projects, My Tasks, Reports, Admin
- `src/api/` placeholder HTTP client and feature API modules
- `src/stores/` app shell store provider for basic UI state
- `.env.example` for `VITE_API_BASE_URL`

## Commands

```bash
npm install
npm run dev
npm run build
npm run lint
```
